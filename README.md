# MissionEngineeringToolbox
The aim of the Mission Engineering Toolbox is to provide modelling and simulation tools to suport use case analysis, system design and assessement of multi-domain RF systems involving platforms, sensors, effectors and C2 systems for radar, EW, comms etc.

Using the framework contained in the US DoD Mission Engineering Guide [https://ac.cto.mil/wp-content/uploads/2023/11/MEG_2_Oct2023.pdf] as the overall guide and inspiration.

Note: all models, algorithms and parameter values are designed for representation of generic systems at UNCLASSIFIED level.
