# Mission Engineering Toolbox
The aim of the Mission Engineering Toolbox is to provide modelling and simulation tools to support assessement of multi-domain RF systems involving platforms, sensors, effectors and C2 systems for radar, EW, comms etc.

Using the US DoD Mission Engineering Guide as the framework for use case analysis, systems thinking and mission assessement and metrics. [https://ac.cto.mil/wp-content/uploads/2023/11/MEG_2_Oct2023.pdf]

*Mission engineering is an interdisciplinary process encompassing the entire technical effort to analyze, design, and integrate current and emerging operational needs and capabilities to achieve desired mission outcomes.*

Note: all models, algorithms and parameter values are designed for representation of generic systems at UNCLASSIFIED level.
